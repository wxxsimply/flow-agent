# 运镜与表演原则（Motion Principles）

与 [camera-language.md](camera-language.md)、[character-performance.md](character-performance.md) 配合使用。

## 单镜结构

1. **预备**：重心稳定、视线方向明确、与环境有接触（脚/手）。
2. **动作**：单一主动作，幅度小，可在约 1 秒内完成。
3. **冲击**（可选）：动作顶点，表情最强点。
4. **反应**：情绪可见变化。
5. **收势**：回到可识别稳定姿态。

压缩为 3 条 `action_beats` 时：预备 / 进行 / 收势+反应。

## 运镜

- 每镜只选一种主运动：推、拉、摇、跟、固定。
- 运镜方向与人物移动方向一致。
- 用**时间顺序**写可见变化，禁止静态海报式描述。

## 跨镜连续性

- 屏幕方向（180 度规则）。
- 服装锁定词逐字重复。
- 光线色调同场景不突变。
