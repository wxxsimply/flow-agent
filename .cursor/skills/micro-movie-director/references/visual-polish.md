# 画面精致度（Visual Polish）

**[扩写]** 光影、色调、材质、氛围；**[分镜]** visual_prompt 精度；**[i2v]** 减少 AI 伪影关键词。

## 1. 光影（Lighting）

### 主光逻辑

- 标明主光源方向（窗左、顶光、逆光轮廓）。
- 硬光：戏剧、 noir、强烈阴影边缘。
- 软光：肤色友好、浪漫、回忆。
- **rim light（轮廓光）**：分离人物与背景，[分镜] 写「轮廓光勾边发丝」。

### 时间与天气

| 条件 | 可视要素 |
|------|----------|
| 黄金时刻 | 暖侧光、长阴影、天空橙粉 |
| 阴天 | 漫射光、低对比、皮肤无硬阴影 |
| 夜戏 | 实用光源（霓虹、路灯）、高光不超过 3 处 |
| 雨夜 | 地面反光、伞面水珠、霓虹倒影 |

**[扩写]** 同一场景 3 段内光线逻辑一致；转场才允许色温跳变。

## 2. 色调与调色（Color Grade）

- **主色 + 点缀色**：如青橙对比、单色蓝调悬疑。
- **肤色保护**：面部不偏绿、不塑料高光。
- **饱和度**：写实微电影偏中低饱和；奇幻题材可略提饱和。
- **[分镜]** 写「整体青冷调，肤色保持自然」「高光略过曝的复古胶片感」。

## 3. 景深与空间（Depth）

- 浅景深：close / 情绪镜，背景 bokeh 圆形或六边形。
- 深景深：wide / 信息镜，前后景皆清晰。
- **空气透视**：远景略雾、对比度降低。

## 4. 材质与细节（Texture）

**[扩写]** 每段至少 3 种可拍摄材质：

- 织物：棉麻褶皱、皮革反光、湿透后贴肤
- 皮肤：汗珠、风吹发丝、胡茬阴影
- 环境：湿沥青、锈铁、玻璃反光、尘土飞扬

**[分镜]** 禁止只写「精致画面」「电影感」等空词。

## 5. 抗 AI 伪影（Anti-Artifact）

写入 visual_prompt 或 forbidden 语义：

- 禁止：多余手指、融化五官、文字乱码、背景重复纹理、物体边缘呼吸变形
- 禁止：塑料皮肤、过度磨皮、HDR 过饱和
- 鼓励：「皮肤毛孔可见」「布料编织纹理」「自然胶片颗粒（轻微）」

## 6. 竖屏 9:16 构图

- 主体垂直居中或上三分，避免头顶大面积死黑。
- 纵深感用前景虚化 + 中景人物 + 远景灯光层次。
- 字幕区预留：画面下方 1/5 避免关键表情。

## Checklist

- [ ] 主光方向与阴影一致
- [ ] 色调用可执行形容词（非「好看」）
- [ ] 至少 3 种材质/纹理
- [ ] 竖屏安全区已考虑
