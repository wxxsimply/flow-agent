# 连续性台账（Continuity Ledger）

**[扩写续写]** 第 2–3 段前对照；**[分镜]** 全镜生成后核对；**[审查]** review 阶段强制检查。

## 1. 台账字段（每集填写）

```markdown
## 场景 A — [地点名]
- 时间/光线：夜雨 / 霓虹湿街
- 轴线：人物面向橱窗，相机在人物右后侧
- 屏幕方向：主角行进默认向左

### 角色：主角
- 锁定词：（逐字复制，全片不变除非换装）
- 发型/妆：
- 服装状态：干燥 → s04 后湿透
- 手持物：无 → s05 透明伞

### 角色：配角B
- 锁定词：
- 最后出现镜号：

### 环境常量
- 招牌/地标：左侧「药房」绿灯招牌
- 天气：中雨，地面积水反光
```

## 2. 跨段扩写（segment 2–3）

续写前检查：

- [ ] 锁定词与 segment 1 末尾一致
- [ ] 光线/天气未无理由变化
- [ ] 受伤/湿身状态已记录

## 3. 跨镜分镜

| 检查项 | 规则 |
|--------|------|
| 服装 | visual_prompt 含相同锁定词 |
| 道具 | 左手杯在 s02 持有则 s03 不可凭空消失 |
| 方向 | 出画左 → 入画逻辑延续 |
| 景别跳变 | wide→close 须有动机（情绪或发现） |
| 旁白 | narration 不重复（DedupeNarrations） |

## 4. 身份漂移（Identity Drift）修复

高发：脸型、发色、夹克色、配饰消失。

修复策略：

1. 从 s01 提取锁定词写回 visual_prompt 开头
2. 降低该镜运镜幅度，减少 BoN 闪烁候选
3. LLM review patch 仅改 visual_prompt / physics，勿改剧情

## 5. 与 guided-animation 阶段映射

| 阶段 | 台账动作 |
|------|----------|
| 概念/扩写 1 | 建立场景 A 块 + 主角锁定词 |
| 扩写 2–3 | 更新服装状态、新场景块 |
| 分镜 | 每镜 ID 对照台账勾选 |
| produce | 不改台账，仅物理选优 |
