# 叙事-镜头关联（Narrative-Shot Linkage）

标签：**[扩写]** 写入 segment_text；**[分镜]** 写入 narrative_beat / brief_excerpt / camera_angle / visual_prompt。

## 1. 叙事节拍与镜头功能

微电影自动分镜须将每镜绑定剧情节拍，常见功能映射：

| 剧情功能 | 推荐景别 | 运镜倾向 | 旁白特征 |
|----------|----------|----------|----------|
| 铺垫/建立 | wide / full | 固定或慢推 | 交代时空、人物处境 |
| 发展/推进 | medium | 跟拍或摇镜 | 动作与关系变化 |
| 揭示/转折 | close | 推近或反打 | 信息点、情绪突变 |
| 高潮/对峙 | close / extreme close | 固定或短推 | 短句、高密度 |
| 收束/余韵 | wide / medium | 拉远或固定 | 情绪回落、点题 |

**[分镜]** 每镜 `narrative_beat` 必填，且须与 `narration` 情绪一致；禁止「情绪高潮镜」却用远景空镜无主体。

## 2. 蒙太奇节奏（自动切分原则）

- 建立镜（wide）约占 15–25%，不宜连续超过 2 镜同景别同运镜。
- 反应镜（close）须紧跟引发情绪的动作镜，形成「动作→反应」对。
- 场景硬切前，[扩写] 须写过渡句（「切至…」「镜头切换至中景…」）；[分镜] `brief_excerpt` 须含该过渡对应原文。
- 10 秒单镜：一个主动作 + 零或一次运镜变化；勿在一镜内塞多个剧情转折。

## 3. 镜头语言服务剧情（反例）

禁止：

- 旁白写悲伤，镜头写「欢快摇镜扫过人群」而无反讽动机。
- `camera_angle` 只写「中景」而无角度与运镜（须三段式：景别 + 角度 + 主运镜）。
- `visual_prompt` 与 `brief_excerpt` 描述不同场景或不同人物外形。

## 4. brief_excerpt 规则

- 从 `shot_language_brief` 原文摘录 1–2 句，逐字或轻微压缩，不可凭空编造。
- 摘录句须能解释本镜为何如此构图/运镜（因果可读）。
- 用户审阅时可修改 excerpt 与 beat，但修改后 `visual_prompt` 须仍与摘录一致。

## 5. camera_angle 三段式模板

```
{景别}，{平视/俯拍/仰拍/过肩/POV}，{固定|推|拉|摇|跟}（{起态→终态}）
```

示例：`中景，平视，缓慢推近（全身至胸部）`

## 6. 情绪-景别速查

| 情绪 | 优先景别 | 光线提示 |
|------|----------|----------|
| 紧张 | medium / close | 侧光、硬阴影 |
| 温馨 | medium | 暖色温、柔光 |
| 悬疑 | close + 留白 | 低调、局部光 |
| 史诗 | wide | 轮廓光、大景深 |

**[扩写]** 情绪变化时须同步调整景别或运镜，并写明理由。
